Github 홈페이지 제작 방법

참고자료
https://brunch.co.kr/@everiwon/42

github에서 홈페이지를 제작하려면 위 링크를 따라 제작하시면 됩니다. 
22.09.01 기준 현재 https://github.com/SNU-BDA-Lab/SNU-BDA-Lab.github.io 에 정상적으로 파일이 올라가 있는 상태여서 특별한 문제가 없는 한 홈페이지를 새로 만들어야 할 일은 없을 것입니다. 다만 혹시 모를 경우를 위해 백업하는 방법을 적어두겠습니다.

참고자료 링크에서 3단계 '홈페이지 파일 업로드' 부분에서 해야 할 일을 설명드리겠습니다. 링크를 보시면 3단계에 이미지처럼 'Add file'에서 'Upload files'를 통해 파일들을 올릴 것입니다. 이것은 페이지를 제작할 때 사용하면 되고 백업을 위해서는 오른쪽은 초록색 버튼 'Code'를 눌러보겠습니다. 그럼 뜨는 메뉴들 중에서 Download ZIP이라는 버튼이 있는데 이 버튼을 눌러 레포지토리에 있는 파일들을 모두 받아두시면 됩니다. 나중에 압축파일을 해제하여 이 파일들을 전부 'Upload files'로 올려주시면 됩니다. 파일 전체 용량은 9.3GB 정도 되니 참고하시길 바랍니다. 

홈페이지 내용을 수정하는 방법은 위의 github 링크에서 'How_To_Use_Github_pages.pdf'를 참고하시면 될 것 같습니다. https://github.com/SNU-BDA-Lab/SNU-BDA-Lab.github.io/blob/main/How_To_Use_Github_pages.pdf

별 내용이 없어 글로 설명을 드렸는데 이것이 빈약할 수도 있습니다. 만약 궁금한 점 있으시면 물어봐주세요!
