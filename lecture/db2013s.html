Database (4190.301): Spring 2013

This is an introductory database course. We will cover the basic topics of database technology, including data models, query languages, integrity constraints, database design, storage structures, and transaction processing.

Course Information

Instructor: Taewhi Lee (twlee AT idb.snu.ac.kr, 301-453, TEL 880-1830)
Class Schedule: Mon, Wed 2:00 ~ 3:15 PM
Class Room: 302-107
Textbook: "Database System Concepts" by Silberschatz, Korth, and Sudarshan. 6th ed., 2010. (McGraw-Hill)
Prerequisites: Computer programming, Data structures
