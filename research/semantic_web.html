Introduction

Semantic Web is a concept of next generation of current Web. It enables sharing web of data and makes web more interoperable. World Wide Web Consortium leads the Semantic Web movement. This movement is now on converting Current Web dominated by unstructured or semi-structured data into structured semantic web data.
